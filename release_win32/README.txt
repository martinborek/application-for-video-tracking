RUNNING THE APPLICATION
  1) Open file video_anonymizer/videoanonymizer.exe.
  2) If the application failed to start due to missing libraries (MSVCP120.dll), install the libraries from vcredist_x86.exe and run the application again (step 1).
     Note: You can download the library package on this website: https://www.microsoft.com/en-US/download/details.aspx?id=40784
